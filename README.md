# Usage Guide - Real Estate Comp Engine

Date: May 7, 2026

This document explains how to use the real estate comp engine in normal use and in edge-case situations. It is written for operational use. It does not assume coding knowledge.

## What This Tool Is For

Purpose:
- Take a subject property address.
- - Pull matching property and comp data from Propelio.
  - - Score nearby comps using lot-size fit, living-area fit, neighborhood match, and boundary context.
    - - Return the top comp set and an ARV summary.
      - - Export the result to Excel.
       
        - Main output file:
        - - `output/comps_report.xlsx`
         
          - If that file is open or locked:
          - - The tool automatically writes a fallback file such as `output/comps_report_YYYYMMDD_HHMMSS.xlsx`
           
            - ## Before You Run It
           
            - Purpose:
            - - Make sure the tool has everything it needs before the first run.
             
              - Input:
              - - Python 3.10+
                - - Installed dependencies from `requirements.txt`
                  - - Propelio username and password
                    - - Optional proxy configuration if used in your environment
                     
                      - Step by step:
                      - 1. Open the project folder.
                        2. 2. Open a terminal in the project folder.
                           3. 3. Create and activate a Python virtual environment.
                              4. 4. Install dependencies from `requirements.txt`.
                                 5. 5. Put Propelio credentials in `.env` or pass them on the command line.
                                    6. 6. Run the command with a full street address.
                                      
                                       7. Standard setup example:
                                      
                                       8. ```bash
                                          python -m venv .venv
                                          .venv\Scripts\activate
                                          pip install -r requirements.txt
                                          python main.py "4044 Williamsburg Rd, Dallas, TX" --username "YOUR_EMAIL" --password "YOUR_PASSWORD"
                                          ```

                                          Expected result:
                                          - Console prints the subject summary, scored comp pool, top 3 comps, neighborhood summary, and ARV breakdown.
                                          - - Excel report is written into the `output` folder.

                                                                                                                       - ## Case 2 - Similar Neighborhood Match
                                                                                                                   
                                                                                                                       - Purpose:
                                                                                                                       - - Use comps from a normalized similar subdivision name when the names are effectively the same but formatted differently.
                                                                                                                        
                                                                                                                         - Input:
                                                                                                                         - - Subject address in a subdivision like `Glenridge Estates 3`
                                                                                                                           - - Nearby comp rows labeled `Glenridge Estates`
                                                                                                                            
                                                                                                                             - Example behavior:
                                                                                                                             - - The tool marks these as `similar`
                                                                                                                               - - Neighborhood multiplier becomes `0.9`
                                                                                                                                 - - Boundary flag remains `no_boundary_penalty` when the comp is a similar match
                                                                                                                                  
                                                                                                                                   - Example output row behavior:
                                                                                                                                   - - `Nbhd Match Type`: `similar`
                                                                                                                                     - - `Neighborhood Match Reason`: `Normalized match (case/spacing)`
                                                                                                                                       - - `Final Confidence`: `0.90` or lower if lot fit is weaker
                                                                                                                                        
                                                                                                                                         - How to use it step by step:
                                                                                                                                         - 1. Run the tool normally.
                                                                                                                                           2. 2. Open `Top Comps` sheet.
                                                                                                                                              3. 3. Look at the `Nbhd Match Type` column.
                                                                                                                                                 4. 4. If the value is `similar`, the comp is being treated as a same-neighborhood normalized match, not as a cross-subdivision fallback.
                                                                                                                                                    5. 5. Confirm the confidence formula row includes `0.9000 nbhd_mult`.
                                                                                                                                                      
                                                                                                                                                       6. ## Case 3 - Living Area Outlier Filter
                                                                                                                                                      
                                                                                                                                                       7. Purpose:
                                                                                                                                                       8. - Prevent large living-area mismatches from entering the selected comp set.
                                                                                                                                                         
                                                                                                                                                          - Input:
                                                                                                                                                          - - Subject property with known living area
                                                                                                                                                            - - Candidate comps with living area far above or below the subject
                                                                                                                                                             
                                                                                                                                                              - Rule:
                                                                                                                                                              - - If living-area variance exceeds 40%, the comp is excluded.
                                                                                                                                                               
                                                                                                                                                                - Example output:
                                                                                                                                                                - - Log lines such as:
                                                                                                                                                                  -   - `Excluded 4164 Saranac Dr ... 4661.0 sqft vs subject 1413.0 sqft (230% variance)`
                                                                                                                                                                      -   - `Excluded 3942 Rochelle Dr ... 4152.0 sqft vs subject 1413.0 sqft (194% variance)`
                                                                                                                                                                          -   - `Excluded 9655 Lakemont Dr ... 2426.0 sqft vs subject 1413.0 sqft (72% variance)`
                                                                                                                                                                           
                                                                                                                                                                              - How to use it step by step:
                                                                                                                                                                              - 1. Run the tool for the subject property.
                                                                                                                                                                                2. 2. Watch the terminal output during scoring.
                                                                                                                                                                                   3. 3. Look for `[SQFT FILTER] Excluded ...` lines.
                                                                                                                                                                                      4. 4. Open the final `Top Comps` sheet.
                                                                                                                                                                                         5. 5. Confirm the excluded outlier properties do not appear in the selected comp rows.

                                                                                                                                                                                            9. Purpose:
                                                                                                                                                                                            10. - Handle a comp where Propelio does not return living area, without hiding that data gap.
                                                                                                                                                                                               
                                                                                                                                                                                                - Input:
                                                                                                                                                                                                - - Subject address where one selected comp has no living area in API output
                                                                                                                                                                                                 
                                                                                                                                                                                                  - Expected behavior:
                                                                                                                                                                                                  - - The comp is not silently filtered.
                                                                                                                                                                                                    - - The comp is marked as missing sqft.
                                                                                                                                                                                                      - - The report discloses the issue clearly.
                                                                                                                                                                                                       
                                                                                                                                                                                                        - Example output:
                                                                                                                                                                                                        - - In `Top Comps`, living area shows:
                                                                                                                                                                                                          -   - `N/A (sqft missing — not filtered)`
                                                                                                                                                                                                              - - Final confidence shows an asterisk:
                                                                                                                                                                                                                -   - `0.90 *`
                                                                                                                                                                                                                    - - `Top Comps` note row says:
                                                                                                                                                                                                                      -   - `Living area missing from API — sqft comparability unverified for this comp (included, not filtered)`
                                                                                                                                                                                                                          - - `ARV Analysis` includes:
                                                                                                                                                                                                                            -   - `Missing Sqft Comps in ARV Pool`
                                                                                                                                                                                                                             
                                                                                                                                                                                                                                - How to use it step by step:
                                                                                                                                                                                                                                - 1. Run the tool normally.
                                                                                                                                                                                                                                  2. 2. Open the `Top Comps` sheet.
                                                                                                                                                                                                                                     3. 3. Check whether any comp has `N/A (sqft missing — not filtered)`.
                                                                                                                                                                                                                                        4. 4. If yes, scroll one row below the data and read the note.
                                                                                                                                                                                                                                           5. 5. Open `ARV Analysis` and check the `Missing Sqft Comps in ARV Pool` row.
                                                                                                                                                                                                                                              6. 6. Treat that comp as usable with disclosure, not as fully verified sqft comparability.
                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                 7. ## Case 5 - Nearby Neighborhood Expansion
                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                 8. Purpose:
                                                                                                                                                                                                                                                 9. - Allow the tool to fill the comp set from nearby neighborhoods when exact subdivision matches are too few.
                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                    - Input:
                                                                                                                                                                                                                                                    - - Subject address
                                                                                                                                                                                                                                                      - - `--expand-radius` flag
                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                        - Example command:
                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                        - ```bash
                                                                                                                                                                                                                                                          python main.py "4044 Williamsburg Rd, Dallas, TX" --username "YOUR_EMAIL" --password "YOUR_PASSWORD" --expand-radius
                                                                                                                                                                                                                                                          ```
                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                          Expected behavior:
                                                                                                                                                                                                                                                          - Exact subdivision matches are preferred first.
                                                                                                                                                                                                                                                          - - If too few exact matches exist, nearby subdivision comps are merged in.
                                                                                                                                                                                                                                                            - - Similar matches remain unpenalized by the expansion multiplier.
                                                                                                                                                                                                                                                              - - Same-city different-subdivision matches may carry lower confidence and boundary flags such as `within_0.3mi` or `major_street_possible`.
                                                                                                                                                                                                                                                               
                                                                                                                                                                                                                                                                - Example console signal:
                                                                                                                                                                                                                                                                - - `Expanding to nearby neighborhoods - only 0 exact matches found`
                                                                                                                                                                                                                                                                 
                                                                                                                                                                                                                                                                  - How to use it step by step:
                                                                                                                                                                                                                                                                  - 1. Add `--expand-radius` to the normal run command.
                                                                                                                                                                                                                                                                    2. 2. Let the run finish.
                                                                                                                                                                                                                                                                       3. 3. Review the console for the expansion warning line.
                                                                                                                                                                                                                                                                          4. 4. Open `Top Comps` and compare these columns:
                                                                                                                                                                                                                                                                             5.    - `Nbhd Match Type`
                                                                                                                                                                                                                                                                                   -    - `Neighborhood Match Reason`
                                                                                                                                                                                                                                                                                        -    - `Boundary Flag`
                                                                                                                                                                                                                                                                                             -    - `Final Confidence`
                                                                                                                                                                                                                                                                                                  - 5. Use the `Confidence Formula` column to understand exactly why a comp scored lower.

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       6. 6.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           7. 7.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              8. 8.
